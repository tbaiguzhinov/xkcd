# Загрузка комиксов xkcd в группу в VK

Код выгружает рандомный комикс с сайта [https://xkcd.com/](https://xkcd.com/) и публикует его на стене группы в VK.
Для корректной работы кода, требуется заранее созданная группа в VK и приложение (зарегистрировать можно [тут](http://dev.vk.com/), у которого будет доступ на размещение публикаций в этой группе.

## Как установить

Python3 должен быть уже установлен.
* Скачайте код
* Установите зависимости  
```pip install -r requirements.txt```
* Запустите программу командой  
```python3 main.py```

## Переменные окружения

Для корректной работы кода необходимо указать переменные окружения. Для этого создайте файл `.env` рядом с `main.py` и запишите туда следующие обязательные переменые:
* `VK_API_KEY` - Токен ключ API VK.
* `VK_GROUP_ID` - Номер группы в VK, можно увидеть в url группы после слова "public".

## Как получить токен ключ API VK

Вставьте в адресную строку браузера url:  
```https://oauth.vk.com/authorize?client_id=0000000&scope=photos,groups,wall,offline&response_type=token```  
где вместо client_id укажите номер ID Вашего приложения (его можно найти в разделе "Редактировать" > "Настройки" в Вашем приложении).
